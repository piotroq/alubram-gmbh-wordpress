## 6. Formy Grzecznościowe

### 6.1 Sie-Form (Formalna)

**ZASADA:** Zawsze używaj formy grzecznościowej "Sie" w komunikacji z klientem.

| Osoba | Forma | Przykład |
|-------|-------|----------|
| 2. os. l.mn. (Sie) | **Sie-form** | "Kontaktieren Sie uns" |
| 2. os. l.poj. (du) | ❌ NIE UŻYWAĆ | "Kontaktiere uns" — BŁĄD! |

**Przykłady poprawne:**
```
✅ "Gerne beraten wir Sie kostenlos."
✅ "Kontaktieren Sie uns für ein Angebot."
✅ "Wir freuen uns auf Ihre Anfrage."
✅ "Haben Sie Fragen? Rufen Sie uns an."
```

**Przykłady niepoprawne:**
```
❌ "Gerne beraten wir dich kostenlos." — zbyt nieformalnie
❌ "Kontaktier uns für ein Angebot." — zbyt nieformalnie
❌ "Wir freuen uns auf deine Anfrage." — zbyt nieformalnie
```

### 6.2 Zaimki i Końcówki

| Element | Forma | Przykład |
|---------|-------|----------|
| Zaimek | **Sie** (wielka litera!) | "für Sie", "mit Sie" |
| Zaimek dzierżawczy | **Ihr** (wielka litera!) | "Ihr Angebot", "Ihre Anfrage" |
| Końcówki czasowników | **-en** | "kontaktieren Sie", "rufen Sie an" |

**Deklinacja zaimka "Sie":**

| Przypadek | Zaimek | Przykład |
|-----------|--------|----------|
| Nominativ | Sie | "Sie sind willkommen" |
| Genitiv | Ihrer | "wir gedenken Ihrer" |
| Dativ | Ihnen | "für Ihnen" |
| Akkusativ | Sie | "wir beraten Sie" |

## 10. Checklist Tłumaczeniowa

### 10.1 Przed Publikacją

```
TREŚĆ
[ ] Wszystkie zdania w Sie-Form (forma grzecznościowa)
[ ] Brak "du" formy w całej treści
[ ] "Sie" i "Ihr" pisane wielką literą
[ ] Długość zdań: max 25 słów
[ ] Brak błędów ortograficznych
[ ] Brak błędów gramatycznych (deklinacja, koniugacja)
[ ] Spójna terminologia (ten sam produkt = ta sama nazwa)

STYL
[ ] Ton profesjonalny, ale przyjazny
[ ] Konkretne dane (liczby, fakty)
[ ] Brak przesadnego entuzjazmu
[ ] Brak pustych sloganów ("najlepszy", "najtańszy")
[ ] Aktywna strona (nie pasywna)
[ ] Korzyści dla klienta podkreślone

SEO
[ ] Słowa kluczowe naturalnie wplecione
[ ] Keyword density: 1.5-2.5%
[ ] Nagłówki H1, H2, H3 z keywordami
[ ] Alt texty obrazów po niemiecku
[ ] Meta title i description po niemiecku

PRAWNE
[ ] Impressum kompletne (wszystkie dane)
[ ] Datenschutzerklärung kompletna
[ ] AGB dostępne
[ ] Cookie consent (DSGVO)
[ ] Formularz z checkbox zgody

LOKALIZACJA
[ ] Waluta: EUR (€)
[ ] Jednostki: mm, cm, m, kg
[ ] Format daty: DD.MM.YYYY
[ ] Format telefonu: +43 (0) XXX XXX XXX
[ ] Adres w formacie austriackim
```
